## 编译说明

### 环境要求

要编译 LaTeX 文档，您需要安装：

1. **TeX Live**（推荐）或 **MiKTeX**
   - Windows: 下载 [TeX Live](https://www.tug.org/texlive/) 或 [MiKTeX](https://miktex.org/)
   - Linux: `sudo apt-get install texlive-full`
   - macOS: `brew install --cask mactex`

2. **必要的 LaTeX 宏包**
   - ctex（中文支持）
   - tikz（绘图）
   - pgfplots（图表）
   - fancyhdr（页眉页脚）
   - 其他常用宏包

### 编译方法

#### 方法一：使用命令行

```bash
# 切换到 course_proposal 目录
cd course_proposal

# 使用 xelatex 编译（推荐，支持中文）
xelatex proposal.tex
xelatex proposal.tex  # 再编译一次以生成目录和交叉引用

# 清理临时文件（可选）
rm -f *.aux *.log *.out
```

#### 方法二：使用 LaTeX 编辑器

推荐使用以下编辑器：
- **Overleaf**（在线编辑器，无需安装）
- **TeXstudio**（跨平台桌面应用）
- **VS Code** + LaTeX Workshop 插件

操作步骤：
1. 在编辑器中打开 `proposal.tex`
2. 选择编译引擎为 `XeLaTeX`（支持中文）
3. 点击编译按钮

#### 方法三：使用在线编译

1. 访问 [Overleaf](https://www.overleaf.com/)
2. 上传 `proposal.tex` 文件
3. 编译器选择 `XeLaTeX`
4. 点击 "Recompile" 按钮

## 研究内容概要

### 研究背景

社会模拟作为计算社会科学的重要分支，致力于通过构建计算模型来理解和预测复杂的社会现象。Casevo 框架结合了大语言模型与多智能体系统，为社会模拟提供了强大的工具。然而，在应对复杂和动态的社会场景时，其智能体的决策机制仍存在改进空间。

### 研究方向

本项目聚焦于**智能体决策能力优化**，具体包括：

1. **多层次推理机制研究**
   - 引入 Tree of Thought (ToT) 等高级推理方法
   - 设计分层决策架构
   - 研究推理深度对决策质量的影响

2. **记忆检索与利用策略优化**
   - 优化基于 ChromaDB 的向量检索算法
   - 设计上下文感知的记忆筛选机制
   - 研究短期记忆与长期记忆的协同利用策略

3. **反思机制优化**
   - 改进 Reflection 算法
   - 引入元认知机制
   - 设计动态反思策略

4. **多智能体协同决策机制**
   - 设计信息交换与决策协商协议
   - 研究群体决策中的共识形成机制
   - 探索分布式决策与集中式决策的权衡策略

### 实验设计

本项目将构建三个严谨的对比实验场景：

1. **选举投票场景**：基于 2020 年美国总统大选辩论内容，配置 101 个具有不同政治倾向的选民智能体，在小世界网络中进行六轮辩论模拟。详细参数包括 LLM 温度 0.7、记忆检索前 5 条、反思阈值 0.6 等。

2. **资源分配场景**：模拟 50 个智能体在资源受限（1000 单位总量）情况下的协商过程，研究智能体如何通过多轮协商达成公平分配方案，最多 10 轮协商。

3. **信息传播场景**：在包含 200 个节点的无标度网络中研究虚假信息扩散，10% 节点初始接收虚假信息，评估不同决策机制对抑制虚假信息的有效性。

每个场景将设置基线组（原始 CoT）、优化组 A（仅 ToT）和优化组 B（全部优化）三组对照实验，每组独立运行 10 次以排除随机因素。

### 预期成果

1. **优化后的 Casevo 框架代码**
   - 增强型思维链模块（`enhanced_chain.py`）
   - 优化的记忆管理模块（`advanced_memory.py`）
   - 协同决策模块（`collaborative_decision.py`）
   - 决策质量评估模块（`decision_evaluator.py`）

2. **详细实验报告**
   - 对比实验数据
   - 数据可视化
   - 案例分析

3. **性能对比分析**
   - 决策质量分析
   - 推理能力分析
   - 计算效率分析
   - 综合评价
